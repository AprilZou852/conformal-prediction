\section{Reproducibility: Source Code}
\label{sec:appendix_code}

This appendix provides the complete source code for reproducing all experiments and figures in the paper. The code is organized into four scripts:

\begin{enumerate}
    \item \texttt{simulation.py} --- Simulation experiments (data generation, model training, Monte Carlo evaluation).
    \item \texttt{realdata.py} --- Real-data experiments (data loading, feature selection, HazardNN ensemble, conformal calibration).
    \item \texttt{simulation\_plots.py} --- Visualization of simulation results (smoothed conditional coverage and width curves, grouped bar plots).
    \item \texttt{plot\_conditional\_coverage.py} --- Conditional coverage scatter plots for real datasets (PC1 quartile grouping).
\end{enumerate}

All code is available at \url{[to be added upon publication]}.
